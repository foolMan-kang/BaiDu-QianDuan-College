1.什么是CSS，CSS是如何工作的
    答：CSS是层叠样式表，通过改变html标签属性，来修饰html标签的外观。

2.CSS的基本语法是怎样的
    答：selector {declaration1; declaration2; ... declarationN } 即选择器{多种属性 ;号分离}

3.CSS选择器是什么概念，简单选择器和属性选择器是什么
    答：用于给指定标签添加属性。
        简单选择器是指使用id，标签名，class等公有属性进行选择。
        属性选择器是指对有特定属性的元素进行选择。

4.文本样式都有哪些相关属性，对应哪些值
    答：font-size，font-family，font-weight，color等。
